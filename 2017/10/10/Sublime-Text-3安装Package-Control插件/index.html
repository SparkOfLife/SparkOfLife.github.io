Sublime Text 3安装Package Control插件

10 Oct 2017

简介

Package Control是一个管理Sublim Text插件的插件，通过Package Control我们可以直接使用Sublime Text控制台来安装其他插件，快捷简单。

安装

方法一：

通过Ctrl+`快捷键或者View > Show Console菜单打开控制台，复制粘贴如下代码回车即可。

若出现网络连接不畅，Package Control网址打不开等意外情况，可使用下面这种方法。

方法二：

  1. Package Control源文件下载

    下载地址：https://github.com/wbond/package_control

    点击右上角的Clone or download按钮，然后选择Download ZIP


  2. 将下载的ZIP文件解压，并重命名为Package Control，未重命名或重命名错误则会出现如下错误提示


  3. 在菜单栏中点击Preferences > Browe Packages...，打开一个文件夹，复制刚才的Package Control文件至此目录下。

  4. 在菜单中点击Preferences，若出现Package Settings和package control选项，就说明安装Package Control成功了。

使用

安装插件

Package Control安装成功后我们就可以用它管理插件了，首先使用快捷键Ctrl+Shift+P（macOS下是Command +Shift+P）
或者点击菜单栏中的Tools > Command Palette...进入Sublime Text命令面板，输入install从列表中选择Install Package回车。
（此时Sublime Text开始请求远程插件仓库的索引，第一次请求会有一点慢，请稍等。）
