LeetCode 1 Two Sum

10月 17, 2017 发布在 LeetCode

题意

题目如下：

Given an array of integers, return indices of the two numbers such that they add up to a specific target.

You may assume that each input would have exactly one solution, and you may not use the same element twice.

原题链接

即给出一个数组，找到和为target的两个数字的位置。

分析

暴力求解复杂度是O(n^2),数组很大时，会LTE。

在python中可以用dict来记录出现的数（key值为数值，value为数组下标），只要判断有没有和当前的数凑成 target 的数，再找出来就行。由于从python字典中取值，其时间复杂度为O(1)，因此这种算法的时间复杂度为O(n)，成功AC。
